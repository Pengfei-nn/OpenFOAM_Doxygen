Detailed Description

Remove a dictionary entry.

The #remove directive takes a list or a single keyType. For example, 

    #remove entry0
    #remove ( entry1 entry2 entry3 otherEntry )
    #remove "entry[1-3]"
    #remove ( "entry[1-3]" otherEntry )
    #remove ^dict1.subdict2.entry2
    #remove "/dict1/subdict2/entry1"
Note
Unless otherwise scoped, the removal occurs in the current context. To remove from other scopes, a dot-scoped or slash-scoped syntax is required. The slash-scoped syntax must be quoted to ensure that it is properly parsed.
