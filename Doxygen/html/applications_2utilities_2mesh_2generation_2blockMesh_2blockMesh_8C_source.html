33 Description
34  A multi-block mesh generator.
35 
36  Uses the block mesh description found in
37  - \c system/blockMeshDict
38  - \c system/<region>/blockMeshDict
39  - \c constant/polyMesh/blockMeshDict
40  - \c constant/<region>/polyMesh/blockMeshDict
41 
42 Usage
43  \b blockMesh [OPTION]
44 
45  Options:
46  - \par -blockTopology
47  Write the topology as a set of edges in OBJ format and exit.
48 
49  - \par -region <name>
50  Specify alternative mesh region.
51 
52  - \par -dict <filename>
53  Alternative dictionary for the block mesh description.
54 
55  - \par -sets
56  Write cellZones as cellSets too (for processing purposes)
57 
58  - \par -noClean
59  Do not remove any existing polyMesh/ directory or files
60 
61  - \par -time
62  Write resulting mesh to a time directory (instead of constant)
63
