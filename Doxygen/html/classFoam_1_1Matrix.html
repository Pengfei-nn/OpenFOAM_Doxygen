Detailed Description

template<class Form, class Type>
class Foam::Matrix< Form, Type >

A templated (m x n) matrix of objects of <T>. The layout is (mRows x nCols) - row-major order: 

    (0,0)
      +---> j [nCols]
      |
      |
      v
      i [mRows]
