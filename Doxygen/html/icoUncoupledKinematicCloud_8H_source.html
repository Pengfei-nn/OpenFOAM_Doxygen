32 Description
33  This functionObject tracks a uncoupled kinematic particle cloud in the
34  specified velocity field of an incompressible flow (laminar, RANS or LES).
35 
36  It may be used in conjunction with any transient single-phase incompressible
37  flow solver such as pisoFoam or pimpleFoam and tracks the particles or
38  parcels without affecting the flow-field.
39 
40  The kinematicCloud requires the density of the fluid which is looked-up
41  from constant/transportProperties dictionary and the acceleration due to
42  gravity which is read from the constant/g file if present or defaults to
43  zero.
44 
45  The kinematicCloud properties are read from the
46  constant/kinematicCloudProperties dictionary in the usual manner.
47 
48 Usage
49  Example of function object specification:
50  \verbatim
51  tracks
52  {
53  libs ("liblagrangianFunctionObjects.so");
54  type icoUncoupledKinematicCloud;
55  }
56  \endverbatim
57 
58  Where the entries comprise:
59  \table
60  Property | Description | Required | Default value
61  type | Type name: icoUncoupledKinematicCloud | yes |
62  U | Name of the velocity field | no | U
63  kinematicCloud | Name of the kinematicCloud | no | kinematicCloud
64  \endtable
