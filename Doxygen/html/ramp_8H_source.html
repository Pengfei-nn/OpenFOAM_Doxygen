29 Description
30  Ramp function base class for the set of scalar functions starting from 0 and
31  increasing monotonically to 1 from \c start over the \c duration and
32  remaining at 1 thereafter.
33 
34  Usage:
35  \verbatim
36  <entryName> <rampFunction>;
37  <entryName>Coeffs
38  {
39  start 10;
40  duration 20;
41  }
42  \endverbatim
43  or
44  \verbatim
45  <entryName>
46  {
47  type <rampFunction>;
48  start 10;
49  duration 20;
50  }
51  \endverbatim
52 
53  Where:
54  \table
55  Property | Description | Required | Default value
56  start | Start time | no | 0
57  duration | Duration | yes |
58  \endtable
59
