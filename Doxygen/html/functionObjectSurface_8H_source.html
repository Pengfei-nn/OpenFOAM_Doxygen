27  Foam::functionObjects::runTimePostPro::functionObjectSurface
28 
29 Description
30  Visualisation of surface data from function object output,
31  typically the result of a prior sampled surfaces operation.
32 
33  Operates in a "live" mode, in which the previously sampled surfaces
34  are retrieved from the functionObject registry.
35  Or in the traditional file-based mode, in which the sampling is used
36  to generate a file and its name is retrieved from the functionObject
37  properties.
38 
39  File-based import is restricted to "vtk" and "vtp" formats.
