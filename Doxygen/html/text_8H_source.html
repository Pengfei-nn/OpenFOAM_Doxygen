27  Foam::functionObjects::runTimePostPro::text
28 
29 Description
30  Define text element for runTimePostProcessing
31 
32  Example of text object specification:
33  \verbatim
34  text1
35  {
36  string "text to display";
37  position (0.1 0.05);
38  size 18;
39  // halign left; // (left | centre | right)
40  bold yes;
41  // Optional entry
42 
43  shadow false;
44 
45  // Optionally override default colour
46  // colour (0 1 1);
47 
48  // Optional entry
49  timeStamp yes; // Append solution time to string
50  }
51  \endverbatim
52 
53  Dictionary controls
54  \table
55  Property | Description | Required | Default
56  string | Text to display | yes |
57  position | The (x y) viewport position | yes |
58  positions | Multiple (x y) viewport positions | no |
59  size | The font size in points | yes |
60  halign | Text justification (left/centre/ right) | no | left
61  bold | Use bold font | yes |
62  italic | Use italic font | no | false
63  shadow | Add text shadow | no | false
64  colour | Override default text colour | no |
65  timeStamp | Append solution timeName to string | no | false
66  \endtable
67 
68  Inherited controls
69  \table
70  Property | Description | Required | Default
71  visible | Display the object | no | true
72  opacity | Object opacity | no | 1.0
73  \endtable
74 
75 Note
76  The string text is expanded on input using stringOps::inplaceExpand()
77  to expand dictionary and environment variable entries.
