Read points.

This is the first thing to do when reading FPMA,FPMB,FLMA files. Return the number of points read.

The file format is as follows: 

* NUMBER_OF_VERTICES
* x0 y0 z0 x1 y1 z1 ... xN-1 yN-1 zN-1
*
