List times using the timeSelector, or use to remove selected time directories.

Original source file foamListTimes.C

Usage
foamListTimes [OPTION]

Options:

  • -processor
    List times from processor0/ directory
  • -rm
    Remove selected time directories
  • -verbose
    Report progress during removal
    Note
    The OpenFOAM banner information is suppressed so that the output can be piped into another command.
