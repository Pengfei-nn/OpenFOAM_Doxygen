27 Class
28  Foam::functionObjects::fieldAverageItem
29 
30 Description
31  Helper class to describe what form of averaging to apply. A set will be
32  applied to each base field in Foam::fieldAverage, of the form:
33 
34  \verbatim
35  {
36  mean on;
37  prime2Mean on;
38  base time; // iteration
39  window 200; // optional averaging window
40  windowName w1; // optional window name (default = "")
41  windowType approximate; // window type
42 
43  allowRestart yes; // optional, used for windowType 'exact'
44  }
45  \endverbatim
46 
47  The averaging window corresponds to the averaging interval (iters or time)
48  If not specified, the averaging is over 'all iters/time'
49 
50  Available window types:
51  - \c none : no windowing
52  - \c exact : exact windowing - additional files will be stored and written
53  - \c approximate : approximate variant that does not store/write additional
54  fields
