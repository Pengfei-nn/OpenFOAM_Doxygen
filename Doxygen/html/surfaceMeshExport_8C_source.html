28  surfaceMeshExport

30 Group
31  grpSurfaceUtilities

33 Description
34  Export from surfMesh to various third-party surface formats with
35  optional scaling or transformations (rotate/translate) on a
36  coordinateSystem.
37 
38 Usage
39  \b surfaceMeshExport outputFile [OPTION]
40 
41  Options:
42  - \par -clean
43  Perform some surface checking/cleanup on the input surface.
44 
45  - \par -name <name>
46  Specify an alternative surface name when writing.
47 
48  - \par -scaleIn <scale>
49  Specify a scaling factor when reading files.
50 
51  - \par -scaleOut <scale>
52  Specify a scaling factor when writing files.
53 
54  - \par -dict <dictionary>
55  Specify an alternative dictionary for constant/coordinateSystems.
56 
57  - \par -from <coordinateSystem>
58  Specify a coordinate system when reading files.
59 
60  - \par -to <coordinateSystem>
61  Specify a coordinate system when writing files.
62 
63 Note
64  The filename extensions are used to determine the file format type.
65 
66 \*---------------------------------------------------------------------------*/
