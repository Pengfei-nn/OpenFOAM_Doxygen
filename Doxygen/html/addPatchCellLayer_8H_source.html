26 Class
27  Foam::addPatchCellLayer
28 
29 Description
30  Adds layers of cells to outside of polyPatch. Can optionally create
31  stand-alone extruded mesh (addToMesh=false).
32 
33  Call setRefinement with offset vector for every patch point and number
34  of layers per patch face and number of layers per patch point.
35  - offset vector should be zero for any non-manifold point and synchronised
36  on coupled points before calling this.
37  - offset vector of zero will not add any points.
38  - gets supplied the number of extruded layers both per face and per
39  point. Usually the point nlayers is the max of surrounding face nlayers.
40 
41  point nlayers:
42  - 0 : no extrusion. Any surrounding face being extruded becomes 'prism'
43  - >0 : should be max of surrounding face nlayers.
44 
45  - differing face nlayers: 'termination' : (e.g. from 2 to 4 layers) match
46  at original patch face side.
47 
48  E.g. 2 boundary faces on patches a,b. 2 layers for a, 3 for b.
49 
50  \verbatim
51  Was:
52 
53  a b <- patch of boundary face
54  +------+------+
55  | | | <- original cells
56  +------+------+
57 
58  Becomes:
59 
60  a b <- patch of boundary face
61  +------+------+
62  + +------+
63  +------+------+
64  +------+------+
65  | | | <- original cells
66  +------+------+
67  \endverbatim
68 
69 
70  - added faces get same patchID as face they are extruded from
71  - 'side' faces (i.e. on the edge of pp) get the patchID/zoneID of the
72  other patch/zone they are connected to (hopefully only 1)
73 
74 
75  E.g. 3 boundary faces on patches a,b. b gets extruded, a doesn't.
76 
77  \verbatim
78  a b b <- patch of boundary face
79  +------+------+------+
80  | | | | <- cells
81  +------+------+------+
82 
83 
84  ^ ^ <- wanted extrusion vector (none at far right)
85  a | b | b <- patch of boundary face
86  +------+------+------+
87  | | | | <- cells
88  +------+------+------+
89 
90  b
91  +------+\ b 1. prism cell added onto second b face since
92  a a| | ----\ only one side gets extruded.
93  +------+------+------+ 2. side-face gets patch a, not b.
94  | | | |
95  +------+------+------+
96  \endverbatim
