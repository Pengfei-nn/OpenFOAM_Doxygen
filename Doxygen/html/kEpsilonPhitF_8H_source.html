27  Foam::RASModels::kEpsilonPhitF
28 
29 Group
30  grpRASTurbulence
31 
32 Description
33  The k-epsilon-phit-f turbulence closure model for incompressible and
34  compressible flows.
35 
36  The model is a three-transport-equation linear-eddy-viscosity turbulence
37  closure model alongside an elliptic relaxation equation:
38  - Turbulent kinetic energy, \c k,
39  - Turbulent kinetic energy dissipation rate, \c epsilon,
40  - Normalised wall-normal fluctuating velocity scale, \c phit,
41  - Elliptic relaxation factor, \c f.
42 
43  Reference:
44  \verbatim
45  Standard model (Tag:LUU):
46  Laurence, D. R., Uribe, J. C., & Utyuzhnikov, S. V. (2005).
47  A robust formulation of the v2-f model.
48  Flow, Turbulence and Combustion, 73(3-4), 169–185.
49  DOI:10.1007/s10494-005-1974-8
50  \endverbatim
51 
52  The default model coefficients are (LUU:Eqs. 19-20):
53  \verbatim
54  kEpsilonPhitFCoeffs
55  {
56  Cmu 0.22, // Turbulent viscosity constant
57  Ceps1a 1.4, // Model constant for epsilon
58  Ceps1b 1.0, // Model constant for epsilon
59  Ceps1c 0.05, // Model constant for epsilon
60  Ceps2 1.9, // Model constant for epsilon
61  Cf1 1.4, // Model constant for f
62  Cf2 0.3, // Model constant for f
63  CL 0.25, // Model constant for L
64  Ceta 110.0, // Model constant for L
65  CT 6.0, // Model constant for T
66  sigmaK 1.0, // Turbulent Prandtl number for k
67  sigmaEps 1.3, // Turbulent Prandtl number for epsilon
68  sigmaPhit 1.0, // Turbulent Prandtl number for phit = sigmaK
69  }
70  \endverbatim
71 
72 Note
73  The name of the original variable replacing 'v2' is 'phi' (LUU:Eq. 14).
74  However, the name 'phi' preexisted in OpenFOAM; therefore, this name was
75  replaced by 'phit' herein.
