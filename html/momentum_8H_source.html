32 Description
33  Calculates linear/angular momentum, reporting integral values
34  and optionally writing the fields.
35 
36  Data is written into momentum.dat in the
37  postProcessing/<functionObjectName> directory.
38 
39 Usage
40  Example of function object specification:
41  \verbatim
42  momentum1
43  {
44  type momentum;
45  libs ("libfieldFunctionObjects.so");
46  ...
47  log yes;
48 
49  regionType all;
50  writeMomentum yes;
51  writePosition yes;
52  writeVelocity yes;
53 
54  cylindrical true;
55 
56  origin (0 0 0);
57  e1 (1 0 0);
58  e3 (0 0 1);
59  }
60  \endverbatim
61 
62  Where the entries comprise:
63  \table
64  Property | Description | Required | Default
65  type | Type name: momentum | yes |
66  log | Log information to standard output | no | no
67  writeMomentum | Write (linear, angular) momentum fields | no | no
68  writePosition | Write angular position component fields | no | no
69  writeVelocity | Write angular velocity fields | no | no
70  p | Pressure field name | no | p
71  U | Velocity field name | no | U
72  rho | Density field name | no | rho
73  rhoRef | Reference density (incompressible) | no | 1.0
74  cylindrical | Use cylindrical coordinates | no | false
75  origin | Origin for cylindrical coordinates | no |
76  regionType | Selection type: all/cellSet/cellZone | no | all
77  name | Name of cellSet/cellZone if required | no |
78  \endtable
79 
80 Note
81  - For incompressible cases, the value of \c rhoRef is used.
82  - When specifying the cylindrical coordinate system, the rotation
83  can be specified directly with e1/e2/e3 axes, or via a \c rotation
84  sub-dictionary
85  For example,
86  \verbatim
87  origin (0 0 0);
88  rotation
89  {
90  type cylindrical;
91  axis (0 0 1);
92  }
93  \endverbatim
