Detailed Description

On-the-fly subset of triSurfaceMesh (based on selected patches)

Note

Example usage: 

spheres_solid1
{
    type subTriSurfaceMesh;

    // Surface to operate on. Avoid duplicate loading
    surface spheres.stl;

    // Regions to operate on (regular expressions allowed)
    patches (solid1);
}
