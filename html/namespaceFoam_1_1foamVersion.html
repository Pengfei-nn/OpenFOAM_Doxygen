Detailed Description

Namespace for OpenFOAM version information.

Version information.

Note
Compile-time version information is conveyed by the OPENFOAM define provided in the wmake rules "General/general". The foamVersion.H file is located directly in the src/OpenFOAM/include directory for easier use by external packages and to allow easier modification during packaging.
