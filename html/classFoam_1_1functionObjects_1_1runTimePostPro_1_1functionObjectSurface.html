Detailed Description

Visualisation of surface data from function object output, typically the result of a prior sampled surfaces operation.

Operates in a "live" mode, in which the previously sampled surfaces are retrieved from the functionObject registry. Or in the traditional file-based mode, in which the sampling is used to generate a file and its name is retrieved from the functionObject properties.

File-based import is restricted to "vtk" and "vtp" formats.
