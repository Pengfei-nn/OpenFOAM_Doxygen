Detailed Description

template<class Type>
class Foam::Function1Types::Scale< Type >

Function1 which scales a given 'value' function by a scalar 'scale' function.

This is particularly useful to ramp a time-varying value by one of the monotonic ramp functions.

Usage for a vector: 

    <entryName>
    {
        type      scale;

        scale
        {
            type        linearRamp;

            start       0;
            duration    10;
        }

        value
        {
            type        sine;

            frequency   10;
            amplitude   1;
            scale       (1 0.1 0);
            level       (10 1 0);
        }
    }

Where:

Property Description Required
value Function of type Function1<Type> yes
scale Scaling function of type Function1<scalar> yes
