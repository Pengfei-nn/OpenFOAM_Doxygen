27  Foam::cellLooper
28 
29 Description
30  Abstract base class. Concrete implementations know how to cut a cell
31  (i.e. determine a loop around the circumference).
32 
33  Loop around the cell is given as the vertices to be cut and edges to
34  be cut (and a weight between 0 and 1 giving where the cut is to be
35  made). Main routine is 'cut' which gets called for every cell and
36  gets the current cut situation and expects to return a loop on the
37  cell circumference.
38 
39  Calling function needs to determine whether cellLooper is compatible with
40  existing set of cuts.
41 
42  Also contains various utility functions which implementations might want to
43  use.
