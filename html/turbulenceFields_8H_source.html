28  Foam::functionObjects::turbulenceFields
29 
30 Group
31  grpFieldFunctionObjects
32 
33 Description
34  Stores turbulence fields on the mesh database for further manipulation.
35 
36  Fields are stored as copies of the original, with the prefix
37  "tubulenceModel:", e.g.:
38 
39  \verbatim
40  turbulenceModel:R
41  \endverbatim
42 
43 Usage
44  Example of function object specification:
45  \verbatim
46  turbulenceFields1
47  {
48  type turbulenceFields;
49  libs ("libfieldFunctionObjects.so");
50  ...
51  fields
52  (
53  R
54  devRhoReff
55  );
56  }
57  \endverbatim
58 
59  Where the entries comprise:
60  \table
61  Property | Description | Required | Default value
62  type | type name: turbulenceFields | yes |
63  fields | fields to store (see below) | yes |
64  \endtable
65 
66  Where \c fields can include:
67  \plaintable
68  k | turbulence kinetic energy
69  epsilon | turbulence kinetic energy dissipation rate
70  omega | turbulence specific dissipation rate
71  nuTilda | turbulence modified viscosity
72  nut | turbulence viscosity (incompressible)
73  nuEff | effective turbulence viscosity (incompressible)
74  mut | turbulence viscosity (compressible)
75  muEff | effective turbulence viscosity (compressible)
76  alphat | turbulence thermal diffusivity (compressible)
77  alphaEff | effective turbulence thermal diffusivity (compressible)
78  R | Reynolds stress tensor
79  devReff | Deviatoric part of the effective Reynolds stress
80  devRhoReff | Divergence of the Reynolds stress
81  L | turbulence length scale
82  I | turbulence intensity
83  \endplaintable
84 
85 See also
86  Foam::functionObjects::fvMeshFunctionObject
87  Foam::functionObjects::timeControl
