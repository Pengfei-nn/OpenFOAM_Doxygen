26 Class
27  Foam::lumpedPointMovement
28 
29 Description
30  The movement \a driver that describes initial point locations, the
31  segmentation for pressure integration, the current state of the
32  points/rotations, and forwarding to the externalFileCoupler
33  communication coordinator.
34 
35  The lumpedPointIOMovement class is simply a registered version of
36  the same.
37 
38  See externalFileCoupler for more information about some of
39  the communication parameters and setup.
40 
41  \heading Dictionary parameters
42  \table
43  Property | Description | Required | Default
44  axis | Reference axis for the locations | yes |
45  locations | List of lumped point locations | yes |
46  centre | Offset of patch points to locations | no | automatic
47  division | Division (0-1) for pressure forces | no | 0
48  relax | Relaxation/scaling for updating positions | no |
49  interpolationScheme | The interpolation scheme | yes |
50  forces | Optional forces dictionary | no |
51  communication | Required communication dictionary | yes |
52  \endtable
53 
54  \heading Parameters for communication dictionary
55  \table
56  Property | Description | Required | Default
57  inputName | Name of positions input file | yes |
58  outputName | Name of forces output file | yes |
59  logName | Name of log file | no | movement.log
60  inputFormat | Input format: dictionary/plain | yes |
61  outputFormat | Output format: dictionary/plain | yes |
62  scaleInput | Input scaling parameter dictionary | no |
63  scaleOutput | Output scaling parameter dictionary | no |
64  \endtable
65 
66  \heading Parameters for optional communication/scaleInput dictionary
67  \table
68  Property | Description | Required | Default
69  length | Scaling for input positions | no | 1
70  \endtable

72  \heading Parameters for optional communication/scaleOutput dictionary
