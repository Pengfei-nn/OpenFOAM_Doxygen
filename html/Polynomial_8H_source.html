27  Foam::Polynomial
28 
29 Description
30  Polynomial templated on size (order):
31 
32  \verbatim
33  poly = sum(coeffs[i]*x^i) + logCoeff*log(x)
34  \endverbatim
35 
36  where <tt> 0 <= i <= N </tt>
37 
38  - integer powers, starting at zero
39  - \c value(x) to evaluate the poly for a given value
40  - \c derivative(x) returns derivative at value
41  - \c integral(x1, x2) returns integral between two scalar values
42  - \c integral() to return a new, integral coeff polynomial
43  - increases the size (order)
44  - \c integralMinus1() to return a new, integral coeff polynomial where
45  the base poly starts at order -1
