Detailed Description

Simple (simplistic) mpi-profiling.

Usage
Example of function object specification: 
profiling
{
    type  parProfiling;
    libs  ("libutilityFunctionObjects.so");

    // Report stats on exit only (instead of every time step)
    executeControl  onEnd;
    writeControl    none;
}
