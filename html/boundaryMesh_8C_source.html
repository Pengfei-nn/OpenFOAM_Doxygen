110  const label startEdgeI,

134

138  }
139

142  // edgeI : first edge on this segment
143  // vertI : one of the endpoints of this segment

854 // one big octree. Problem was that faces on top of each other, differing

856 // find only one. We could detect that it was probably finding the wrong one

858 // one (since they occupy exactly the same space)
