33 Description
34  This function object calculates the flux across selections of faces.
35 
36  Output comprises, per set of faces, the fluxes:
37  - positive
38  - negative
39  - net
40  - absolute
41 
42 Usage
43  \verbatim
44  fluxSummary1
45  {
46  type fluxSummary;
47  libs ("libfieldFunctionObjects.so");
48  ...
49  write yes;
50  log yes;
51  mode cellZoneAndDirection;
52  cellZoneAndDirection
53  (
54  (porosity (1 0 0))
55  );
56  scaleFactor 1.2;
57  }
58  \endverbatim
59 
60  Where the entries comprise:
61  \table
62  Property | Description | Required | Default
63  type | Type name: fluxSummary | yes |
64  write | Write flux data to file | no | yes
65  log | Write flux data to standard output | no | yes
66  mode | Mode to generate faces to test | yes |
67  scaleFactor | Optional factor to scale result | no | 1
68  tolerance | Tolerance for the reference direction | no | 0.8
69  phi | Surface flux field | no | phi
70  \endtable
71 
72  The mode is one of:
73  - faceZone
74  - faceZoneAndDirection
75  - cellZoneAndDirection
76  - surface
77  - surfaceAndDirection
78 
79  Output data is written to files of the form <timeDir>/<faceZoneName>.dat
80 
81 Note
82  For surface and direction, can use phi = 'U' for determining the fluxes.
83 
84 See also
85  Foam::functionObjects::fvMeshFunctionObject
86  Foam::functionObjects::writeFile
87  Foam::functionObjects::timeControl
