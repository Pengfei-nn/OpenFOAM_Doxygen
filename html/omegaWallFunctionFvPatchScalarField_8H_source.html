27 Class
28  Foam::omegaWallFunctionFvPatchScalarField
29 
30 Group
31  grpWallFunctions
32 
33 Description
34  This boundary condition provides a wall constraint on specific turbulent
35  kinetic energy dissipation rate, i.e. \c omega, for low- and high-Reynolds
36  number turbulence models.
37 
38  The near-wall omega may be either blended between the viscous region and
39  logarithmic region values using:
40 
41  \f[
42  \omega = sqrt(\omega_{vis}^2 + \omega_{log}^2)
43  \f]
44 
45  where
46 
47  \vartable
48  \omega_{vis} | omega in viscous region
49  \omega_{log} | omega in logarithmic region
50  \endvartable
51 
52  see eq.(15) of:
53  \verbatim
54  Menter, F., Esch, T.
55  "Elements of Industrial Heat Transfer Prediction"
56  16th Brazilian Congress of Mechanical Engineering (COBEM),
57  Nov. 2001
58  \endverbatim
59 
60  or switched between these values based on the viscous-to-turbulent \c y+
61  value derived from \c kappa and \c E.
62 
63 Usage
64  \table
65  Property | Description | Required | Default value
66  beta1 | Model coefficient | no | 0.075
67  blended | Blending switch | no | false
68  \endtable
69 
70  Example of the boundary condition specification:
71  \verbatim
72  <patchName>
73  {
74  // Mandatory entries
75  type omegaWallFunction;
76 
77  // Optional entries
78  }
79  \endverbatim
80 
81 Note
82  The coefficients \c Cmu, \c kappa, and \c E are obtained from
83  the specified \c nutWallFunction in order to ensure that each patch
84  possesses the same set of values for these coefficients.
85 
86  Some tests have shown that the stepwise switching method provides
87  more accurate predictions for 10 < y+ < 30 when used with high Reynolds
88  number wall-functions.
89 
90  In addition, the stepwise switching method provides accurate predictions
91  when used with continuous wall-functions.
92 
93 See also
94  Foam::fixedInternalValueFvPatchField
95  Foam::epsilonWallFunctionFvPatchScalarField
