27  Foam::phasePressureModel
28 
29 Description
30  Particle-particle phase-pressure RAS model
31 
32  The derivative of the phase-pressure with respect to the phase-fraction
33  is evaluated as
34 
35  g0*min(exp(preAlphaExp*(alpha - alphaMax)), expMax)
36 
37  The default model coefficients correspond to the following:
38  \verbatim
39  phasePressureCoeffs
40  {
41  preAlphaExp 500;
42  expMax 1000;
43  alphaMax 0.62;
44  g0 1000;
45  }
46  \endverbatim
