28  Foam::smoothDelta
29 
30 Description
31  Smoothed delta which takes a given simple geometric delta and applies
32  smoothing to it such that the ratio of deltas between two cells is no
33  larger than a specified amount, typically 1.15.

235  //- Fill changedFaces (with face labels) and changedFacesInfo

237  // This is the initial set of faces from which to start the waves.
238  // Since there might be lots of places with delta jumps we can follow
239  // various strategies for this initial 'seed'.
240  // - start from single cell/face and let FaceCellWave pick up all
241  // others from there. might be quite a few waves before everything
242  // settles.
243  // - start from all faces. Lots of initial transfers.
