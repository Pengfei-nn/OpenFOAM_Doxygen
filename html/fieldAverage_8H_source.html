27 Class
28  Foam::functionObjects::fieldAverage
29 
30 Group
31  grpFieldFunctionObjects
32 
33 Description
34  Calculates average quantities for a user-specified selection of volumetric
35  and surface fields.
36  With the %subRegion option, also supports fields on functionObject
37  surface output (eg, sampledSurfaces).
38 
39 
40  Fields are entered as a list of sub-dictionaries, which indicate the type of
41  averages to perform, and can be updated during the calculation. The current
42  options include:
43  - \c mean: arithmetic mean
44  \f[
45  \overline{x} = \frac{1}{N}\displaystyle\sum\limits_{i=0}^N x_i
46  \f]
47  - \c prime2Mean: prime-squared mean
48  \f[
49  \overline{x'}^2 = \frac{1}{N}\displaystyle\sum\limits_{i=0}^N
50  (x_i - \overline{x})^2
51  \f]
52  - \c base: average over 'time', or 'iteration' (\f$N\f$ in the above)
53  - \c window: optional averaging window, specified in 'base' units
54 
55  Average field names are constructed by concatenating the base field with
56  the averaging type, e.g. when averaging field 'U', the resultant fields
57  are:
58  - arithmetic mean field, \c UMean
59  - prime-squared field, \c UPrime2Mean
60 
61  Information regarding the number of averaging steps, and total averaging
62  time are written on a per-field basis to the
63  \c "<functionObject name>Properties" dictionary, located in <time>/uniform
64 
65  When restarting form a previous calculation, the averaging is continuous or
66  may be restarted using the \c restartOnRestart option.
67 
68  The averaging process may be restarted after each calculation output time
69  using the \c restartOnOutput option or restarted periodically using the \c
70  periodicRestart option and setting \c restartPeriod to the required
71  averaging period.
72 
73 Usage
74  \verbatim
75  fieldAverage1
76  {
77  type fieldAverage;
78  libs ("libfieldFunctionObjects.so");
79 
80  writeControl writeTime;
81 
82  restartOnRestart false;
83  restartOnOutput false;
84  periodicRestart false;
85  restartPeriod 0.002;
86 
87  fields
88  (
89  U
90  {
91  mean on;
92  prime2Mean on;
93  base time;
94  windowType exact;
95  window 10.0;
96  windowName w1;
97  }
98  p
99  {
100  mean on;
101  prime2Mean on;
102  base time;
103  }
104  );
105  }
106  \endverbatim
107 
108  Where the entries comprise:
109  \table
110  Property | Description | Required | Default
111  type | Type name: fieldAverage | yes |
112  restartOnRestart| Restart the averaging on restart | no | no
113  restartOnOutput | Restart the averaging on output | no | no
114  periodicRestart | Periodically restart the averaging | no | no
115  restartPeriod | Periodic restart period | conditional |
116  restartTime | One-shot reset of the averaging | no | great
117  fields | list of fields and averaging options | yes |
118  subRegion | Name for alternative objectRegistry | no | ""
119  \endtable
120 
121 
122 Note
123  To employ the \c prime2Mean option, the \c mean option must be selecetd.
