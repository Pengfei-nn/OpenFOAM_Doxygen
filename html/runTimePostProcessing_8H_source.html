39 Description
40  Generate images during run-time.
41 
42  The functionality makes use of the VTK libraries (see http://www.vtk.org)
43  which provide a broad set of functionality for scene composition and
44  manipulation.
45 
46  Images are generated using a combination of function object output
47  and additional data - eg, surfaces and text.
48  Current capabilities include support for:
49  - Camera
50  - Objects
51  - Points
52  - Lines
53  - Surfaces
54  - Scalar bars
55  - Annotations
56  - Selection of colour maps
57  .
58 
59  Mandatory entries:
60  \verbatim
61  output
62  {
63  name image;
64  width 800;
65  height 600;
66  }
