28  Foam::functionObjects::regionSizeDistribution
29 
30 Group
31  grpFieldFunctionObjects
32 
33 Description
34  Creates a size distribution via interrogating a continuous phase fraction
35  field.
36 
37  Looks up a phase-fraction (alpha) field and splits the mesh into regions
38  based on where the field is below the threshold value. These
39  regions ("droplets") can now be analysed.
40 
41  Regions:
42  - print the regions connected to a user-defined set of patches.
43  (in spray calculation these form the liquid core)
44  - print the regions with too large volume. These are the 'background'
45  regions.
46  - (debug) write regions as a volScalarField
47  - (debug) print for all regions the sum of volume and alpha*volume
48 
49  Output (volume scalar) fields include:
50  - alpha_liquidCore : alpha with outside liquid core set to 0
51  - alpha_background : alpha with outside background set to 0.
52 
53  %Histogram:
54  - determine histogram of diameter (given minDiameter, maxDiameter, nBins)
55  - write graph of number of droplets per bin
56  - write graph of sum, average and deviation of droplet volume per bin
57  - write graph of sum, average and deviation of user-defined fields. For
58  volVectorFields these are those of the 3 components and the magnitude.
59  - (optional) write graph of histogram of centroids on iso planes
60  downstream of the injector determined by origin, direction and maxDiameter
61  up to maxDownstream
62 
63 Usage
64  Example of function object specification:
65  \verbatim
66  regionSizeDistribution1
67  {
68  type regionSizeDistribution;
69  libs ("libfieldFunctionObjects.so");
70  ...
71  field alpha;
72  patches (inlet);
73  threshold 0.4;
74  fields (p U);
75  nBins 100;
76  maxDiameter 0.5e-4;
77  minDiameter 0;
78  setFormat gnuplot;
79  origin (0 0 0);
80  coordinateRoation
81  {
82  type cartesian;
83  e3 (0 1 1);
84  e1 (1 0 0);
85  }
86 
87  // Optional downstream iso-plane bins.
88  isoPlanes true;
89 
90  // Plane normal and point definition
91  direction (1 0 1);
92  origin (1e-4 0 5e-4);
93 
94  // Maximum diameter of the cylinder formed by the origin point
95  // and direction
96  maxDiameter 3e-4;
97 
98  // Maximum downstream distance
99  maxDownstream 6e-4;
100 
101  // Number of iso-plane bins
102  nDownstreamBins 20;
103 
104  }
105  \endverbatim
106 
107  Where the entries comprise:
108  \table
109  Property | Description | Required | Default value
110  type | type name: regionSizeDistribution |yes|
111  field | phase field to interrogate | yes |
112  patches | patches from which the liquid core is identified | yes|
113  threshold | phase fraction applied to delimit regions | yes |
114  fields | fields to sample | yes |
115  nBins | number of bins for histogram | yes |
116  maxDiameter | maximum region equivalent diameter | yes |
117  minDiameter | minimum region equivalent diameter | no | 0
118  setFormat | writing format | yes |
119  origin | origin of local coordinate system | yes |
120  coordinateRoation | orientation of local coordinate system | no
121  log | Log to standard output | no | yes
122  isoPlanes | switch for isoPlanes | no | false
123  origin | origin of the plane when isoPlanes is used | no | none
124  direction | direction of the plane when isoPlanes is used | no | none
125  maxDiameter | maximum diameter of the sampling cylinder when
126  isoPlanes is used | no | none
127  nDownstreamBins | number of bins when isoPlanes is used | no | none
128  maxDownstream | maximum distance from origin when isoPlanes is used
129  | no | none
130  \endtable
